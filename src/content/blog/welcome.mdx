This is a template post. Copy this file to a new `.md` or `.mdx` in `src/content/blog/`, rename it, edit the frontmatter, and replace this content with your own.

## Frontmatter fields

| Field | Required | Notes |
| --- | --- | --- |
| `title` | yes | Post title (shown in the listing and at the top of the page) |
| `description` | yes | Used in `<meta>` tags for SEO and social sharing |
| `pubDate` | yes | Any date string |
| `updatedDate` | no | Shown next to publication date |
| `heroImage` | no | Optional image at the top of the post |
